# VentaExpress | Data Cleaning and Sales Analysis

**Tool used:** Excel / Google Sheets

## Project overview
This project analyzes fourth-quarter 2024 sales data for VentaExpress, an e-commerce business focused on technology products in Mexico and Colombia.

The work included data cleaning, standardization, business metric calculation, and spreadsheet-based visual analysis to transform raw sales data into an executive-friendly report.

## Business objective
The goal of this project was to answer key commercial questions:

- Which product sold the most units?
- Which city generated the highest sales?
- What was the best-performing month?
- What insights can support better business decisions?

## Tools used
- Excel / Google Sheets
- Data cleaning
- Spreadsheet formulas
- Business metrics
- Charts and visual analysis

## Key results
- Total sales: **2,944,620.61**
- Average sale per transaction: **3,910.52**
- Total transactions: **753**
- Top-selling product: **Laptop-Oficina-32GB**
- Best-performing city: **Monterrey**
- Best-performing month: **November**
